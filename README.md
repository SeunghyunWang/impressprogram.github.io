# 공동 작업하기

- Download Zip으로 파일 다운로드
- gem install jekyll
- gem install jekyll-bundler
- gem install jektll-paginate
- jekyll new [폴더이름]
- cd [폴더이름]
- 안에 파일 전부 삭제하고 Download Zip으로 받은 파일 집어넣기
- bundle install
- (로컬에서 테스트로 돌려보려면) bundle exec jekyll serve
- (서버에 제출하려면) git에 올리기
